======================================================
DRISHYAMITRA - NEW COMPUTER SETUP & RUN INSTRUCTIONS
======================================================

If you are downloading this project on a new computer, follow these steps to rebuild the environment.

STEP 1: DOWNLOAD THE CODE
Open a terminal and run:
git clone https://github.com/moin08s/Drishyamitra.git
cd Drishyamitra

STEP 2: ADD SECRET ENV FILES (Not on GitHub for security)
1. In the "backend" folder, create a file named ".env" and add:
   SECRET_KEY=my_super_secret_key
   DATABASE_URL=sqlite:///app.db
   GROQ_API_KEY=your_groq_api_key
   GMAIL_SENDER=your_email@gmail.com
   GMAIL_PASSWORD=your_16_letter_app_password

2. In the "frontend" folder, create a file named ".env" and add:
   VITE_API_URL=http://127.0.0.1:5000/api

STEP 3: REBUILD & START FRONTEND
1. Open a terminal in the "frontend" folder.
2. Run: npm install
3. Run: npm run dev

STEP 4: REBUILD & START BACKEND
1. Open a new terminal in the "backend" folder.
2. Run: python -m venv venv
3. Activate it: 
   - Windows: venv\Scripts\activate
   - Mac/Linux: source venv/bin/activate
4. Run: pip install -r requirements.txt
5. Run: python app.py

STEP 5: OPEN THE APP
Go to your browser and open http://localhost:5173
